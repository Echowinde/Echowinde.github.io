我开始写博客啦

发表于 2022-05-02

从去年发现 GitHub Pages 可以托管静态页面之后，我就萌生了搭建个人博客网站的想法，但是因为学习和科研一直很忙就是懒，所以直到这个被关在学校里的五一节，在 Apex 坐牢无数把，B站刷到索然无味后，我终于把这个博客搭建起来了。目前这个博客使用 Hexo 框架部署在 GitHub Pages 上，很多功能和美化我都还没有做，博客也还没有独立的域名，等更新稳定了再去买一个吧。之后我会把生活、科研和学习的日常所思所想 post 在这上面，顺便用 hexo-douban 这个插件把豆瓣的电影和图书记录同步过来，这样也会有些影评之类的内容，争取多多更新~
